Algorithm 1: Exact Algorithm

Description

The Exact Algorithm is a brute-force approach designed to find the densest subgraph within a given graph for a specified number of nodes, denoted by h. This method systematically explores all possible subsets of size h from the vertex set and computes the density for each subset. The subset with the highest density is selected as the output.

The algorithm works as follows:

  • Enumerate all possible combinations of h vertices from the graph.
  • For each subset, calculate the number of edges within the induced subgraph.
  • Compute the density as the ratio of the number of edges to the maximum possible edges for h vertices.
  • Track and update the subset with the maximum density found.

Due to its exhaustive search nature, the Exact Algorithm guarantees the optimal solution. However, its time complexity is O(nh) where n is the total number of vertices in the graph. This makes it impractical for large graphs or large values of h, but it performs well for small datasets where precision is crucial.

This algorithm is particularly useful as a benchmark to evaluate the performance and accuracy of heuristic or approximation algorithms for the densest subgraph problem.
