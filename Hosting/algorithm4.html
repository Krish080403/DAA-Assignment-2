Algorithm 4: Core Exact

Description

The CoreExact algorithm is an exact method designed to compute the Core Density Subgraph (CDS) for a given graph G(V, E) with respect to a weight function Ψ(VΨ, EΨ). It aims to find a subset of vertices and edges that maximizes a density function based on both the graph structure and the weight criteria.

The algorithm works as follows:

  • Perform a preliminary decomposition of the input graph to simplify the search space.
  • Identify a (k', Ψ')-core subgraph using pruning techniques to narrow down potential candidates.
  • Initialize sets and parameters to manage vertex subsets and track the maximum density achieved so far.
  • Iteratively process each connected component of the (k', Ψ')-core:
    • For each component with sufficient weight, build a flow network and compute a minimum s-t cut.
    • If the resulting subset S is non-empty, enter a refinement loop to further maximize density.
    • Continuously update the parameter u and recompute minimum cuts to explore denser subgraphs.
    • If a better density is found, update the candidate subgraph accordingly.
  • Finally, compare the obtained density with the current best and update the final output if improved.

The algorithm intelligently combines core decomposition, flow-based cuts, and iterative refinement to guarantee an exact and optimal solution. Although computationally intensive for large graphs, it is highly effective for medium-sized graphs where exactness is required.

CoreExact also leverages pruning and intermediate checks to reduce redundant computations, making it significantly faster than naïve enumeration-based exact methods.
